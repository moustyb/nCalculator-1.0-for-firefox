# nCalculator-1.0-for-kids  
A simple, beautiful calculator built with HTML, CSS, and JavaScript — packaged as a Firefox WebExtension.

---

## Overview
**nCalculator-1.0-for-kids** is a lightweight, local calculator designed for all ages.  
It runs completely offline and does **not collect or transmit any personal data**.  
The extension provides a clean, modern interface optimized for keyboard and touch use.

---

## Features
- **Clean, modern design** with smooth gradient buttons  
- **Responsive layout** – adapts to all screen sizes  
- **Keyboard support** – type numbers or operators directly  
- **Error handling** – invalid or malformed expressions show “Error” instead of crashing  
- **Percentage (%) calculations** supported  
- **Decimal protection** – prevents multiple decimal points in a single number  
- **Safe evaluation** – no `eval()` or remote code execution; uses an internal parser

---

## How to Use
1. Click the **nCalculator** icon in the Firefox toolbar.  
2. Use on-screen buttons or your keyboard to enter calculations.  
3. Press `=` or the `Enter` key to compute results.  
4. Press **AC** to clear the display.  
5. Press **⌫** (Backspace) to delete the last character.  
6. Works entirely offline – no Internet connection needed.
